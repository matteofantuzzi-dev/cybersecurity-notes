# Linux Basics

## Cos'è Linux
Linux is an open-source operating system widely used in servers, networking, and cybersecurity.

## Perché è importante
Many security tools run on Linux, and understanding it is essential for penetration testing and network administration.

## Comandi principali
```bash
pwd       # Shows current directory
ls        # Lists files and folders
cd <dir>  # Change directory

## Errori comuni
